For pool swim activities: change pool size unit to metric, so you can see your stats in metres.

Enable or disable auto-lock, for newer devices like Fenix 8 where the setting has been removed, but the feature *apparently* still exists.

This is useful if:

  • You've restored settings from a backup where auto-lock is enabled and you want to disable it
  • You want to enable auto-lock

Notes:

  • I don't own a Fenix 8, so I can't guarantee this will work, but there's a good chance (see following points)
  • A Garmin forum user has reported that restoring a backup from an older device to Fenix 8 causes the auto-lock feature to be enabled on their Fenix 8. Presumably this is happening because the older device had auto-lock enabled in settings
  • It's reasonable to assume that Fenix 8 still has the auto-lock functionality under the covers, even though the setting has been removed
